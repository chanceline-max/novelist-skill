# 题材适配器

先使用通用工作流，再选择一个适配器。适配器调整读者承诺、冲突发动机和节奏，不替代因果、人物和连续性检查。

## 男频升级/玄幻/系统

- 承诺：能力、地位、资源或认知持续升级，并获得可验证回报。
- 发动机：目标差距 -> 规则/资源限制 -> 选择与代价 -> 升级 -> 更高层敌人。
- 每个升级必须改变可解决的问题，同时制造更大责任或新对手；避免“换地图重复打怪”。
- 单元模板：触发任务、展示规则、第一次受挫、发现隐藏条件、付出代价、反击、战果与新悬念。

## 都市/职业/现实题材

- 承诺：在真实制度、职场或社会关系中获得成长、尊严或情感回报。
- 冲突来自利益、规则、声誉、时间和关系，不要只靠坏人降智。
- 专业细节服务选择：每段行业知识都应改变策略或代价；无法影响行动的资料删掉。

## 悬疑/推理

- 承诺：问题可解、线索公平、真相会重估此前事件。
- 建立“真相表”：真相、读者已知、主角已知、嫌疑人误导、每条线索首次出现与回收位置。
- 反转必须回溯可解释，不能凭新证据临时改规则；每章至少推进一个问题并生成一个更窄的问题。

## 言情/女频关系驱动

- 承诺：关系变化带来持续情绪回报，人物在爱与自我之间做选择。
- 每次暧昧、冲突、和解都要改变关系状态；外部事件应迫使双方暴露需求或底线。
- 甜虐节奏用“靠近 -> 误解/代价 -> 选择 -> 暂时兑现 -> 更深需求”循环，避免只靠误会拖延。

## 无限流/副本

- 承诺：每个副本有独立谜题/规则，同时推进主线和角色关系。
- 副本卡固定包含：入口条件、胜利条件、失败代价、隐藏规则、资源限制、队伍矛盾、结算奖励、主线影响。
- 新副本至少改变一项规则或关系；结算不能只发装备，要带来新债务、线索或选择。

## 历史/架空/世界观厚重题材

- 承诺：时代规则会真实约束人物，选择产生长期后果。
- 维护制度、技术、地理、称谓、时间线表；优先让设定通过行动和利益冲突显现。
- 重大历史节点不可当作免费背景，必须说明主角知道什么、能改变什么、改变会牺牲什么。

## 现实情感/成长

- 承诺：情绪真实、选择有后果、变化可观察。
- 用具体行为替代“他很痛苦”；每个阶段设置可见目标、失败和关系反馈，避免连续独白。
