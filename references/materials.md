# 素材库.md 规范与模板

素材库只保存可转化为场景、选择或细节的材料，不保存没有用途的百科摘抄。每条素材标明来源和可用边界，避免把资料说明直接塞进正文。

## 素材卡

```markdown
### M001｜素材名称
- 类型：职业/地点/物件/制度/动作/感官/意象/真实资料
- 来源与版权备注：
- 可服务的题材/场景：
- 能制造的冲突或选择：
- 可观察细节：声音、气味、触感、操作步骤
- 不能夸大的部分：
- 已使用章节：
```

## 转场规则

资料进入正文前必须经过“人物目标—当下阻力—可见行动”转换。例如医学资料不写成课程，而写成角色在时间压力下判断症状、承担误判后果；职业资料不写成名词列表，而写成一次会影响客户、收入或声誉的决定。
