# 人物.md 规范与模板

人物文件只记录会影响行动和剧情的内容。外貌、星座、喜欢的颜色等资料只有在会影响选择或辨识度时才保留。

## 核心人物卡

```markdown
### 人物：
- 身份/社会位置：
- 表层目标：
- 深层需求：
- 恐惧与伤口：
- 错误信念：
- 价值底线：
- 可越过的底线及代价：
- 秘密：谁知道？谁误解？
- 初始资源/弱点：
- 说话节奏、词汇、常用动作：
- 当前情绪与知识边界：
- 关键关系：
- 三次被迫选择：
- 成长转折：
- 最终选择与结局代价：
```

## 出场检查

每次出场至少完成一项：推进目标、制造阻力、改变关系、提供带代价的信息、展示独特选择。配角不能只负责夸主角、解释设定、递送任务或在高潮时被动挨打；反派必须有自己的判断、计划、资源和失败后的调整。
